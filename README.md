# Gerador de Landing Pages
Projeto feito através da formação Programação com IA da DevClub. Criamos um Gerador de Páginas para diferentes negócios.
Basta você inserir a sua chave da Groq/Open AI para testar e descrever o seu negócio.(ex: Escritório de Topografia que oferece
serviços como Georeferenciamento, CAR, etc, localizado no centro de SP.)
